# Agent Skills for TODO/FIXME Tracker

This directory contains agent skills for working with the TODO/FIXME tracker standard.

## Available Skills

| Skill | Purpose |
|-------|---------|
| `scan-todos.md` | How to scan a codebase for TODO/FIXME items |
| `interpret-artifacts.md` | How to read and understand tracker artifacts |
| `validate-compliance.md` | How to validate TODO/FIXME format compliance |

## Usage

AI agents can use these skills to:
- Discover pending work in a codebase
- Validate that TODOs are properly tracked
- Generate reports on technical debt
- Enforce issue reference requirements

## Integration

These skills work with the `.todo-tracker/` artifacts generated by the scanner:

```
.todo-tracker/
├── manifest.toml    # Scan metadata
├── items.json       # All TODO/FIXME items
└── summary.json     # Statistics
```Agents should read `items.json` to understand what work is pending and whether items are properly tracked with GitHub issues.
